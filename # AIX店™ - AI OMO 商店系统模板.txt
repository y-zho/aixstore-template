## 🚀 What is this?

AIX店™ is a ready-to-use web storefront powered by AI and optimized for the OMO (Online-Merge-Offline) era. This template allows you to instantly deploy a bilingual AI-assisted shopping page using GitHub Pages.

---

## 🔑 Features

- 🌐 **English + 中文** bilingual support  
- 🤖 GPT-based assistant link for customer service  
- 🛒 One-page storefront with product showcase  
- 📎 Easy integration with Stripe/PayPal  
- 📄 Notion-based product management template included  
- 🎨 Fully customizable with your brand logo and text

## 🧰 How to Use

1. Fork or use this template (`Use this template` button)
2. Upload your product info to Notion or fill into static HTML
3. Edit your `index.html` with product data
4. Deploy via GitHub Pages (`Settings → Pages`)
5. Share your link & start selling!
